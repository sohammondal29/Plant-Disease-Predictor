Plant Disease Predictor is an web application that detects plant diseases from leaf images.
Built using TensorFlow Lite and Streamlit, the system analyzes uploaded images, identifies the plant and disease, estimates severity, and generates a downloadable diagnosis report.

The project demonstrates the application of deep learning in agriculture, helping users quickly diagnose plant health and take preventive action.

Link : https://plant-disease-predictor-tgsh.onrender.com/
